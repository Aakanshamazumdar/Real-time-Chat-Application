# 💬 Realtime Chat App

A real-time chat application built using **React**, **Node.js**, **Socket.IO**, and **Express**.

## 🚀 Features

- 🔒 User login (optional)
- 💬 Real-time messaging with WebSockets (Socket.IO)
- 📱 Responsive UI (React)
- 🌓 Dark/light theme toggle
- 🔔 Notification sound
- 😄 Emoji support
